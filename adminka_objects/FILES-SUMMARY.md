### 3. `direct.js` (~600 строк)
**JavaScript логика для страницы Яндекс Директ**

Содержит:
- Инициализацию приложения
- Работу с LocalStorage (сохранение/загрузка конфигурации)
- Загрузку данных через API
- Форматирование чисел и валюты
- Управление состоянием (state machine)
- Функции для показа/скрытия блоков
- Обработку фильтров (период, устройство, кампания)
- Сортировку таблицы
- Поиск по таблице
- Отображение и обновление карточек статистики
- Отправку ошибок и уведомлений
- Public API (функции, доступные из HTML)

**Public API:**
- `init()` — инициализация
- `saveCredentials()` — сохранение учетных данных
- `showAuthForm()` — показать форму авторизации
- `disconnect()` — отключиться от Яндекс Директ
- `handlePeriodChange()` — обработка смены периода
- `handleCustomDateRange()` — применение произвольного диапазона дат
- `handleFilterChange()` — обработка смены фильтров
- `sortTable(column)` — сортировка таблицы
- `refreshData()` — обновить данные

**Архитектура:** Vanilla JavaScript (IIFE паттерн)

**Путь:** `/adminka_objects/direct.js`

### 4. `api-direct.php` (~300 строк)
**Backend API для работы с Яндекс Директ**

Содержит:
- Обработку POST запросов от frontend
- Работу с API Яндекс Директ (OAuth, Bearer токен)
- Получение списка кампаний
- Получение статистики кампаний
- Обмен кода авторизации на токен
- Валидацию токена
- Обработку ошибок
- Использование cURL для HTTP запросов

**Поддерживаемые действия (action):**
- `getCampaigns` — получить список кампаний и их статистику
- `exchangeCode` — обменять код авторизации на токен
- `validateToken` — проверить валидность токена

**Требования:**
- PHP 7.4+
- curl extension
- HTTPS

**Путь:** `/adminka_objects/api-direct.php`

## 🚀 Следующие шаги после установки

1. **Создать OAuth приложение**
   - https://oauth.yandex.ru/client/new
   - Получить Client ID и Client Secret

2. **Первый запуск**
   - Открыть https://ваш-домен.by/adminka_objects/direct.html
   - Ввести Client ID и Client Secret
   - Нажать "Сохранить учетные данные"

3. **Авторизация**
   - Нажать "Подключить"
   - Разрешить доступ (перенаправление на сервер Яндекса)
   - Дождаться загрузки статистики

4. **Использование**
   - Просмотр статистики кампаний
   - Анализ показателей
   - Фильтрация данных
