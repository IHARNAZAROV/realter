# 📚 Яндекс Директ для Админки — Главный индекс

## 🎯 Что это?

Модуль интеграции с **Яндекс Директ** для панели администратора сайта **turko.by**.

Позволяет в удобном формате:
- 📊 Просматривать статистику рекламных кампаний
- 📈 Анализировать показы, клики, CTR, расходы
- 🔍 Искать и фильтровать данные
- 📋 Сравнивать производительность кампаний
- 💾 Сохранять конфигурацию локально

## 🚀 Быстрый старт

### 1️⃣ Первая установка (2-3 минуты)

Откройте админку:
```
https://турко.by/adminka_objects/direct.html
```

Вы сразу увидите инструкцию по подключению прямо на странице.

### 2️⃣ Создание OAuth приложения (3-5 минут)

Перейдите на:
```
https://oauth.yandex.ru/client/new
```

Следуйте инструкциям на странице админки (они там есть!).

### 3️⃣ Ввод учетных данных (1 минута)

Введите полученные:
- Client ID
- Client Secret
- Логин вашего Яндекс.ID

### 4️⃣ Авторизация (30 секунд)

Нажмите **"Подключить"** и разрешите доступ.

### ✅ Готово!

Данные будут загружены автоматически.

## 🔐 Безопасность

### ✅ Безопасно:
- Данные хранятся локально в браузере
- Client Secret не отправляется на сервер (кроме API запросов)
- HTTPS обязателен
- Можно отключиться в любо момент

### ⚠️ Рекомендации:
- Используйте на личном компьютере
- Не делитесь Client Secret
- Регулярно проверяйте активные приложения в Яндексе
- Используйте отдельный Яндекс.ID для админки

## ❓ Часто задаваемые вопросы

**Q: Где мой логин кабинета?**  
A: Это ваш Яндекс.ID для входа в Яндекс.Директ (например, `olga-turko`)

**Q: Где я вижу Client ID и Secret?**  
A: После создания OAuth приложения на https://oauth.yandex.ru/

**Q: Где сохраняются мои данные?**  
A: В локальном хранилище браузера (LocalStorage) на вашем компьютере

**Q: Могу ли я поделиться доступом?**  
A: Нет, используйте на личном компьютере. Для общего доступа нужна другая интеграция.

**Q: Как обновить данные?**  
A: Автоматически при открытии или нажав 🔄 кнопку

**Q: Что если я забыл Client Secret?**  
A: Создайте новое приложение или обновите Secret в существующем

**Q: Какие данные отправляются на сервер?**  
A: Только токен для запросов к API Яндекс Директ, все остальное локально
