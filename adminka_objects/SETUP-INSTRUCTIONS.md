# 🚀 Первичная настройка интеграции Яндекс Директ

## Быстрый старт (5 минут)

### Шаг 1️⃣ Откройте админку Яндекс Директ

Для начала нам нужно создать OAuth-приложение. Перейдите на:
👉 https://oauth.yandex.ru/client/new

### Шаг 2️⃣ Создайте новое приложение

Заполните форму:

**Название приложения:**
```
Админка рекламных кампаний
```

**Описание (опционально):**
```
Интеграция для управления статистикой Яндекс Директ из админки сайта
```

**На вкладке "API Яндекса":**
- Поставьте ✅ рядом с **Яндекс Директ API**

**Redirect URI (очень важно!):**
```
https://турко.by/adminka_objects/direct.html
```

⚠️ ВНИМАНИЕ: используйте **HTTPS**, не HTTP!

### Шаг 3️⃣ Сохраните приложение

Нажмите "Создать приложение"

После создания вы увидите:
- **Client ID** — скопируйте и сохраните
- **Client Secret** — скопируйте и сохраните

## Шаг 4️⃣ Введите учетные данные в админку

1. Откройте админку: https://турко.by/adminka_objects/direct.html
2. Вы увидите блок "Инструкция по подключению"
3. Вставьте туда:
   - **Client ID** (из OAuth приложения)
   - **Client Secret** (из OAuth приложения)
   - **Логин кабинета** (ваш Яндекс.ID, например: `olga-turko`)
4. Нажмите "Сохранить учетные данные"

✅ Данные сохранятся в браузер (локально)

## Шаг 5️⃣ Авторизуйтесь

1. Нажмите кнопку "Подключить"
2. Вы будете переведены на сервер Яндекса
3. Нажмите **"Разрешить"**, чтобы дать приложению доступ к вашему кабинету
4. Вернётесь обратно в админку

⚠️ После авторизации страница автоматически загрузит статистику ваших кампаний

## Готово! 🎉

Теперь вы можете:
- ✅ Смотреть статистику всех кампаний
- ✅ Анализировать показы, клики, CTR, расход
- ✅ Сортировать данные
- ✅ Искать кампании
- ✅ Фильтровать по периодам и устройствам

## ❓ Часто задаваемые вопросы

### Где мой логин кабинета?

Это ваш Яндекс.ID, который вы используете для входа в Яндекс.Директ.
Обычно выглядит как `имя-фамилия` или `email@yandex.ru`

Найти его можно:
1. Откройте https://direct.yandex.ru
2. Нажмите на свой профиль в верхнем углу
3. Посмотрите "Пользователь" или "Логин"

### Что такое Client ID и Client Secret?

Это уникальные идентификаторы вашего приложения, которые позволяют ему получить доступ к API Яндекса.

- **Client ID** — публичный идентификатор
- **Client Secret** — приватный ключ (не делитесь с никем!)

### Как часто обновляются данные?

По умолчанию данные обновляются:
- ✅ При открытии страницы
- ✅ При нажатии кнопки "🔄 Обновить"
- ✅ При смене фильтров (для таблицы)

Облако получает данные за последние 7 дней (можно изменить в фильтрах).

### Я забыл Client Secret

Нужно создать новое приложение:
1. Перейдите https://oauth.yandex.ru/
2. Нажмите на ваше приложение "Админка рекламных кампаний"
3. Нажмите "Редактировать" → "Обновить Client Secret"
4. Вставьте новый Secret в админку

### Ошибка "Invalid redirect URI"

Это значит, что Redirect URI не совпадает. Проверьте:
- ✅ Используете HTTPS (не http://)
- ✅ Путь точно: `/adminka_objects/direct.html`
- ✅ Домен совпадает с вашим сайтом
- ✅ Нет лишних слэшей в конце

### Где сохраняются мои данные?

Все учетные данные сохраняются **локально в вашем браузере**:
- На компьютере, где вы авторизовались
- В папке данных браузера (LocalStorage)
- Не отправляются на сервер (кроме запросов к API Яндекса)

⚠️ Поэтому:
- ✅ Используйте на личном компьютере
- ❌ Не используйте на зал компьютерах
- ❌ Не делитесь доступом к браузеру

### Как отключиться?

В карточке "Статус подключения" нажмите **"Отключить"**.
Все данные авторизации будут удалены.

### Проблема с API Яндекса

Если видите ошибку "API недоступен":
1. Проверьте интернет
2. Откройте https://api.direct.yandex.com/ в браузере
3. Если там ошибка — это проблема со стороны Яндекса

## 🔒 Безопасность

### ⚠️ Важные правила

1. **Client Secret**
   - Это как пароль для приложения
   - Никому не давайте
   - Храните его в безопасности

2. **Отдельный Яндекс.ID**
   - Рекомендуется использовать отдельный аккаунт для админки
   - Давайте ему только доступ к API Директ
   - Не используйте личный аккаунт

3. **Регулярно проверяйте**
   - Список авторизованных приложений
   - Удаляйте неиспользуемые

### Как проверить активные приложения?

1. Откройте https://oauth.yandex.ru/
2. Нажмите на "Приложения с доступом к аккаунту"
3. Там вы видите все приложения
4. Можете удалить доступ в любое время

## 🎓 Дополнительно

### Понимание статистики

**Показы** — количество раз, когда объявление было показано
**Клики** — сколько раз пользователь кликнул на объявление
**CTR** — Click Through Rate = (Клики / Показы) × 100%
**Расход** — сколько денег потрачено на эту кампанию

### Пример

Если кампания за день:
- Показы: 1000
- Клики: 50
- CTR: 5%
- Расход: 500 BYN

Это значит, что 1000 раз объявление показалось, 50 раз люди на него кликнули, и это стоило 500 рублей.

### Как улучшить CTR?

- ✅ Улучшите текст объявления
- ✅ Добавьте релевантные ключевые слова
- ✅ Используйте расширения объявления
- ✅ Проверьте целевую аудиторию
