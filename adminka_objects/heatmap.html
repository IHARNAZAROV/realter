    Данные посетителей (кто / куда / когда)

    Clarity Live Insights показывает агрегированную статистику сессий, а не персональные данные конкретных людей. Ниже — сводка по сессиям, источникам и разрезам (страны, устройства, страницы, время), если API их вернул.

    Готово к проверке.
    • 500 — чаще всего не настроен CLARITY_API_TOKEN на сервере.
    • 429 — превышен лимит запросов Clarity, подождите 1–2 минуты.
    • 502 — временная ошибка сети между сервером и Clarity API.
