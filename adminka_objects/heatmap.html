Heatmap / Карты поведения

Microsoft Clarity блокирует встраивание панели проекта через iframe (заголовок X-Frame-Options: sameorigin), поэтому карта не может отобразиться внутри админки.

Открывайте Clarity в новой вкладке или подключайте API (через backend), если нужны кастомные метрики внутри админки.
