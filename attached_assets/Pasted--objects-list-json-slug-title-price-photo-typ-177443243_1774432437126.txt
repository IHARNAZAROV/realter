Как работает обновление — ключевой момент
Нужен один дополнительный скрипт split-objects.cjs по аналогии с уже существующим generate-sitemap.cjs:

node split-objects.cjs
Он читает мастер-файл objects.json и генерирует все производные файлы. Запускать его нужно после каждого сохранения объекта в панели управления.

Можно сделать двумя способами:

Вариант А — вручную (как generate-sitemap):
После редактирования объектов запускаете в терминале:

node split-objects.cjs && node generate-sitemap.cjs
Вариант Б — автоматически через save.php:
После успешного сохранения save.php вызывает exec("node split-objects.cjs") — пересборка происходит моментально в фоне.
