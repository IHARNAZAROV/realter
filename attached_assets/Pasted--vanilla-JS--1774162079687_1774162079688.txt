Создай страницу “Задачи” для админ-панели сайта недвижимости (vanilla JS, без фреймворков). Это часть единой админ-системы.

📌 Общая идея

Страница задач — это внутренний инструмент управления задачами (как mini Trello / Linear), интегрированный в админку объектов.

Задачи должны:

сохраняться в localStorage
не теряться (backup + восстановление)
быть привязаны к текущей странице сайта (window.location.pathname)
поддерживать доски (boards)
🧠 Архитектура
Хранилище:

Используй localStorage:

ключ: "todo_system_vX"
backup: "todo_system_vX_backup"

Структура:
{
boards: [
{
id: string,
name: string,
tasks: [
{
id: number,
text: string,
completed: boolean,
priority: "normal" | "important" | "urgent",
category: "ui" | "seo" | "bug" | "ads",
page: string
}
]
}
],
activeBoard: string
}

Добавь:

try/catch при загрузке
fallback на backup
защита от null/битых данных
🔗 Интеграция с админкой

Добавь навигацию:

[ Объекты ] [ Задачи ]

текущая страница подсвечена
быстрый переход между страницами
единый стиль header
🎨 UI (очень важно)

Стиль: тёмный, современный SaaS (Linear / Notion)

Верхняя панель:
выбор доски (select)
кнопка "+ Доска"
справа: поиск (маленький input)
Основной блок:
большой input “Добавить задачу”
справа:
выбор приоритета (custom select)
выбор категории (custom select)
кнопка “Добавить”
📋 Список задач

Каждая задача — карточка:

[checkbox] Текст задачи [priority] [category] [✕]

Требования:

hover эффект (подъём + glow)
кастомный checkbox
при completed:
текст перечёркнут
opacity ниже
⚙️ Логика
Добавление:
Enter или кнопка
очищает input
сохраняет в localStorage
Удаление:
по кнопке ✕
безопасно (проверка id)
Toggle:
клик по checkbox
меняет completed
🔄 Поведение
фильтр по текущей странице (page)
поиск по тексту
сортировка по приоритету:
urgent → important → normal
🧲 Drag & Drop
можно менять порядок задач
сохраняется порядок в state
не ломается при фильтре
🔐 Надёжность

Обязательно:

защита от null в массиве задач
проверка id перед операциями
двойное сохранение (backup)
sync между вкладками (storage event)
beforeunload → save()
🎬 UX детали

Добавь:

hover эффект (gradient glow)
микроанимации
фокус на input при загрузке
