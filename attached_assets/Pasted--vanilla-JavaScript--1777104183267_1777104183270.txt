У меня есть админ-панель сайта (vanilla JavaScript, без фреймворков).

Задача:
Создать инструмент (UI-блок) "Генератор JSON объекта недвижимости".

Контекст:
У меня есть шаблон JSON объекта (я приложу его отдельно).
Пример структуры включает:
- вложенные объекты (location)
- массивы (images, features, infrastructure)
- nullable поля (null)
- строки, числа и булевые значения

Что нужно реализовать:

1. UI:
Создать на странице админки блок с двумя полями:

— Поле 1 (input):
textarea, куда я вставляю обычный текст (описание объекта)

— Поле 2 (output):
textarea (readonly), куда автоматически генерируется JSON объект

Также добавить кнопку:
"Сгенерировать JSON"

2. Логика генерации:

После нажатия на кнопку:
- взять текст из первого поля
- на его основе сформировать JSON строго по моему шаблону

Правила:
- структура должна полностью соответствовать шаблону
- все отсутствующие данные → null
- массивы (images, features и т.д.) → если нет данных, то []
- вложенные объекты должны сохраняться (например location)

3. Очистка данных:

Перед генерацией:
- удалить лишние пробелы
- удалить HTML-теги (если есть)
- убрать двойные переносы строк
- привести текст к нормальному виду

4. Генерация JSON:

- все поля из шаблона должны присутствовать
- не добавлять лишние поля
- не менять названия ключей
- строки — в кавычках
- числа — без кавычек
- null — без кавычек

5. Дополнительно (обязательно):

Добавить кнопку:
"Скопировать JSON"

Которая:
- копирует результат из output поля в буфер обмена

6. UX:

- подсветка ошибок (если input пустой)
- уведомление "Скопировано"
- авто-скролл output поля вверх после генерации

7. Важно:

- не использовать никакие фреймворки
- чистый vanilla JS
- код должен быть читаемый и разбит на функции
- не ломать существующую админку

8. Бонус (если возможно):

- вынести шаблон JSON в отдельный объект
- генерацию сделать через функцию mapTemplate(data, template)

Результат:

- я вставляю текст
- нажимаю кнопку
- получаю полностью готовый JSON объект по моему шаблону
- могу сразу скопировать и вставить в базу
