A11Y-01: Аккордеон FAQ использует <a> без href вместо <button>
Файл: faq.html

<h4 class="acod-title">
  <a>
    Сколько времени занимает продажа?
    <span class="indicator"></span>
  </a>
</h4>
<a> без href — не является интерактивным элементом. Недоступно для клавиатурной навигации и скринридеров.
Исправление:

<h4 class="acod-title">
  <button type="button" aria-expanded="false" class="acod-toggle">
    Сколько времени занимает продажа?
    <span class="indicator" aria-hidden="true"></span>
  </button>
</h4>
A11Y-02: Alt-текст логотипа — просто "logo" на всех внутренних страницах
Файлы: blog.html, blog-detail.html, faq.html, nedvizhimost-lida.html, rieltor-lida.html, contact.html, object-detail.html

<img src="images/logo-light.svg" class="site-logo site-logo--light" alt="logo" />
Исправление:

<img src="images/logo-light.svg" class="site-logo site-logo--light" alt="Ольга Турко — Риэлтер в Лиде" />
A11Y-03: Иконки соцсетей в футере не имеют aria-label или текстового содержимого
Файлы: все страницы с футером

<a href="viber://..." class="fa-brands fa-viber"></a>
Это пустые ссылки с иконкой через CSS class. Скринридер прочитает URL.
Исправление:

<a href="viber://..." class="fa-brands fa-viber" aria-label="Написать в Viber"></a>
A11Y-04: Отсутствует skip-link на всех страницах
Ни на одной странице нет skip-навигации для клавиатурных пользователей.
Добавить первым элементом в <body>:

<a href="#main-content" class="skip-link">Перейти к основному содержимому</a>
CSS:

.skip-link {
  position: absolute;
  top: -40px;
  left: 0;
  background: #006064;
  color: white;
  padding: 8px;
  z-index: 10000;
  transition: top 0.2s;
}
.skip-link:focus { top: 0; }
A11Y-05: Галерея в sidebar blog-detail имеет пустые alt
Файл: blog-detail.html

<img src="/images/about-slider/2.webp" alt="" />
Если картинки декоративны — alt="" правильно. Но если они несут смысл — нужны описательные alt.

A11Y-06: Избранное-кнопка div вместо button
Файл: js/filters.js

<div class="favorite-btn ..." data-slug="..." aria-label="...">
  <i class="fa-regular fa-heart"></i>
</div>
<div> не интерактивен нативно. Клавиатурная навигация через Tab не попадёт на него.
Исправление:

<button type="button" class="favorite-btn ..." data-slug="..." aria-label="..." aria-pressed="false">
  <i class="fa-regular fa-heart" aria-hidden="true"></i>
</button>
A11Y-07: aria-expanded не обновляется для аккордеона FAQ
Файл: js/optimize.js, функция handleAccordion()
Аккордеон открывается/закрывается, но aria-expanded не обновляется. Скринридеры не знают о состоянии.

A11Y-08: Форма "Нет результатов" не имеет label
Файл: js/filters.js

<input type="text" id="noResultsContact" placeholder="Telegram @username или телефон">
Нет <label for="noResultsContact">. Placeholder не замена label.
Исправление:

<label for="noResultsContact" class="sr-only">Telegram или телефон</label>
<input type="text" id="noResultsContact" placeholder="Telegram @username или телефон">
