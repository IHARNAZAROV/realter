Нужно доработать страницу blog.html.

Задача:
Создать справа фиксированный sidebar с интерактивным календарем публикаций блога.

Требования к sidebar:
- Sidebar располагается справа от основного контента страницы blog.html.
- На десктопе ширина sidebar примерно 320–360px.
- Sidebar должен оставаться видимым при прокрутке страницы (position: sticky, top: 100px).
- На планшете переносить sidebar под список статей.
- На мобильных устройствах отображать календарь в виде раскрывающегося блока над списком статей.

Календарь:
- Использовать собственную реализацию на vanilla JavaScript без библиотек и фреймворков.
- Календарь должен отображать текущий месяц с возможностью переключения между месяцами стрелками «влево / вправо».
- Формат дней недели: Пн, Вт, Ср, Чт, Пт, Сб, Вс.
- Первый день недели — понедельник.
- Заголовок календаря: месяц и год.
- Календарь должен автоматически анализировать blog.json (или существующий массив статей) и отмечать даты, в которые есть статьи.

Предположить, что у каждой статьи есть поле:
"date": "2026-04-12"

Логика:
- Если в определенную дату опубликована хотя бы одна статья, соответствующая ячейка календаря должна:
  - иметь отдельный корпоративный цвет;
  - отображать маленькую точку или индикатор;
  - менять курсор на pointer;
  - иметь hover-эффект.

При клике по дате:
- Открывать красивый popup / modal рядом с календарем или по центру экрана.
- Popup должен быть стилизован под корпоративный стиль сайта.
- Если в выбранную дату несколько статей, вывести их списком.
- Для каждой статьи в popup показать:
  - миниатюру изображения;
  - заголовок;
  - категорию;
  - краткую дату;
  - кнопку или ссылку «Читать статью».

Пример структуры popup:
- Фото статьи
- Заголовок статьи
- Категория и дата
- Кнопка перехода на article.html?id=...

Поведение popup:
- Плавное появление через opacity + translateY.
- Закрытие по:
  - клику вне popup;
  - нажатию Esc;
  - кнопке закрытия ×.
- Если popup не помещается справа от sidebar, автоматически открывать его поверх календаря по центру экрана.

Корпоративный стиль:
- Использовать стиль, соответствующий дизайну сайта:
  - тёмно-синий / графитовый фон;
  - акцентный золотистый или янтарный цвет;
  - скругления 16–20px;
  - мягкие тени;
  - полупрозрачный glass-effect;
  - тонкие светлые границы.
- Sidebar должен выглядеть как премиальный блок агентства недвижимости.
- Использовать плавные hover-анимации и transition 0.3s.
- Активная дата должна подсвечиваться.
- Текущая дата должна иметь отдельное выделение.

Необходимо:
1. Добавить HTML-разметку sidebar в blog.html.
2. Добавить полный CSS для sidebar, календаря и popup.
3. Создать отдельный JS-модуль calendar-sidebar.js.
4. В JS:
   - получить статьи из существующего массива или blog.json;
   - сгруппировать статьи по датам;
   - отрисовать календарь;
   - обработать переходы между месяцами;
   - открывать popup со статьями.
5. Не использовать сторонние библиотеки.

Пример ожидаемой структуры HTML:

<aside class="blog-calendar-sidebar">
    <div class="calendar-card">
        <div class="calendar-header">
            <button class="calendar-prev"></button>
            <div class="calendar-title"></div>
            <button class="calendar-next"></button>
        </div>

        <div class="calendar-weekdays"></div>
        <div class="calendar-grid"></div>
    </div>
</aside>

<div class="calendar-popup hidden">
    <div class="calendar-popup-content"></div>
</div>

Также необходимо:
- Предусмотреть, что статьи могут загружаться асинхронно.
- Не ломать существующую верстку страницы blog.html.
- Использовать BEM-именование классов.
- Все стили и скрипты должны быть полностью готовы к вставке в проект без дополнительных правок.
- Код должен быть чистым, подробно прокомментированным и адаптивным.
- Для каждой статьи использовать существующие поля объекта:
  - id
  - title
  - image
  - category
  - date
  - slug или ссылка

Дополнительно:
Сделать анимацию, при которой при наведении на дату с публикациями слегка увеличивается ячейка и появляется мягкое свечение. ✨
