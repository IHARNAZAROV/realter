# ЗАДАЧА

Необходимо реализовать на главной странице сайта современный слайдер команды риэлтеров (8 карточек), полностью в стилистике текущего сайта.

Проект написан на vanilla JavaScript без фреймворков.

---

# ОСНОВНЫЕ ТРЕБОВАНИЯ

## 1. Слайдер команды на главной странице

Создать новую секцию команды риэлтеров.

Требования:

* использовать Swiper (актуальную версию проекта)
* 8 карточек риэлтеров
* автопрокрутка
* бесконечный loop
* поддержка touch/swipe
* корректная адаптация под мобильные устройства
* поддержка keyboard navigation
* lazy loading изображений
* плавные hover-эффекты

---

# ВИЗУАЛЬНАЯ СТИЛИСТИКА

Слайдер должен:

* полностью соответствовать текущему UI сайта
* использовать корпоративные цвета сайта
* использовать существующую дизайн-систему
* повторять стили карточек, уже присутствующих на сайте
* НЕ выбиваться из общей визуальной концепции

Перед реализацией:

1. Проанализировать существующие карточки на сайте
2. Использовать похожие:

   * border-radius
   * тени
   * typography
   * spacing
   * hover effects
   * анимации
3. Не создавать новый визуальный стиль

---

# СОДЕРЖИМОЕ КАРТОЧКИ

Каждая карточка должна содержать:

* фотографию риэлтера
* имя и фамилию

* краткое описание/специализацию
* кнопку "Подробнее"

* ссылки на соцсети (иконки)

# ЗАГРУЗКА ДАННЫХ

Не использовать hardcode карточек в HTML.

Необходимо:

* загружать team.json динамически
* генерировать карточки через JavaScript
* реализовать обработку ошибок загрузки JSON
* предусмотреть fallback если фото отсутствует
