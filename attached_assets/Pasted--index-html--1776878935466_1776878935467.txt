Тебе нужно переработать ТОЛЬКО визуальную часть главной страницы (index.html) моего сайта.

ВАЖНО:
- НЕ изменять структуру HTML
- НЕ менять порядок блоков
- НЕ трогать JS и функционал
- НЕ удалять элементы
- НЕ добавлять новые блоки
- Работать только со стилями (CSS)

ЦЕЛЬ:
Привести ВСЕ элементы главной страницы к единому корпоративному стилю, который уже используется на других страницах сайта.

ОСНОВНЫЕ ЗАДАЧИ:

1. УНИФИКАЦИЯ СТИЛЕЙ
- Все кнопки привести к единому стилю (цвет, радиус, hover, активное состояние)
- Все заголовки (h1-h4) привести к единой типографике
- Все тексты — к единому размеру, межстрочному интервалу и цвету
- Все карточки (если есть) — к единому стилю (тень, скругления, отступы, фон)

2. ЦВЕТОВАЯ СИСТЕМА
- Использовать только корпоративные цвета сайта
- Убрать случайные или отличающиеся оттенки
- Привести фон секций к согласованной системе (светлый / акцентный / нейтральный)

3. ОТСТУПЫ И СЕТКА
- Выровнять вертикальные и горизонтальные отступы между блоками
- Привести внутренние отступы секций к единому стандарту
- Обеспечить одинаковую логику spacing (например 8px / 16px / 24px / 32px)

4. КНОПКИ И CTA
- Все CTA должны выглядеть одинаково
- Один основной стиль кнопки (primary)
- Один вторичный стиль (secondary, если есть)

5. ФОРМЫ (если есть)
- Привести input, textarea, select к одному стилю
- Одинаковая высота, рамки, фокус-состояние

6. ИКОНКИ И ГРАФИКА
- Привести размеры и цвет иконок к единому виду
- Убрать несоответствия по стилю (например разные толщины линий)

7. ЭФФЕКТЫ
- Унифицировать hover-эффекты
- Добавить плавные анимации (transition 0.2–0.3s)
- Убрать резкие или выбивающиеся эффекты

8. КАРТОЧКИ И БЛОКИ
- Сделать единый стиль:
  - border-radius
  - box-shadow
  - padding
  - background

9. НЕ ТРОГАТЬ:
- HTML структуру
- ID и классы, используемые в JS
- Любую бизнес-логику

10. РЕЗУЛЬТАТ:
- Обновленный CSS (или SCSS, если используется)
- При необходимости — минимальные правки классов в HTML, НО без изменения структуры

ДОПОЛНИТЕЛЬНО:
Ориентируйся на уже существующие страницы сайта как на источник истины по дизайну.
Главная страница должна выглядеть как логическое продолжение остальных страниц.

КРИТЕРИЙ УСПЕХА:
Страница должна выглядеть как единый цельный продукт, без ощущения, что блоки сделаны в разное время.
