## 📋 Быстрый чеклист для проверки

### Перед началом тестирования

- [ ] Откройте DevTools (F12) → Console
- [ ] Скопируйте и загрузите тестовый скрипт из `COOKIE_CONSENT_TEST_UTILS.js`
- [ ] Или просто скопируйте функцию `fullDiagnostics()` и выполните её

### Основные проверки

#### 1. Чтение куки после перезагрузки
```javascript
getCookieData()
```
**Ожидаемый результат:**
```json
{
  "necessary": true,
  "analytics": true,    // ✅ ВСЕГДА true
  "marketing": false,   // зависит от выбора
  "updatedAt": "2026-03-28T..."
}
```

#### 2. Проверка что analytics не может быть отключена
```javascript
checkAnalyticsAlwaysEnabled()
```
**Ожидаемый результат:** 
```
✅ ВЕРНО: analytics = true (ВСЕГДА включена)
```

#### 3. Проверка UI элементов
```javascript
checkButtons()
checkToggles()
```
**Ожидаемый результат:** Все кнопки найдены, Analytics toggle disabled и checked

#### 4. Полная диагностика всех функций
```javascript
fullDiagnostics()
```
**Ожидаемые результаты:**
- ✅ Куки читаются правильно
- ✅ Все кнопки присутствуют
- ✅ Analytics toggle disabled и checked
- ✅ Яндекс.Метрика загружена
- ✅ window.__applyCookieConsent определена

## 🧪 Сценарии тестирования

### Сценарий 1: Первый визит → Принять всё

1. Выполните в консоли: `deleteCookie()` → `location.reload()`
2. должно появиться окно согласия
3. Нажмите "Принять всё"
4. Проверьте: `getCookieData()`
5. **Ожидается:** `analytics: true`, `marketing: true`

### Сценарий 2: Первый визит → Отклонить

1. Выполните в консоли: `deleteCookie()` → `location.reload()`
2. Должно появиться окно согласия
3. Нажмите "Отклонить"
4. Проверьте: `getCookieData()`
5. **Ожидается:** `analytics: true`, `marketing: false` ⚠️ **ВАЖНО: analytics ВСЕГДА true!**

### Сценарий 3: Первый визит → Закрыть (X)

1. Выполните в консоли: `deleteCookie()` → `location.reload()`
2. Должно появиться окно согласия
3. Нажмите крестик (X) в углу
4. Проверьте: `getCookieData()`
5. **Ожидается:** `analytics: true`, `marketing: false` ⚠️ **ВАЖНО: analytics ВСЕГДА true!**

### Сценарий 4: Открыть настройки и изменить Marketing

1. Нажмите "Настроить"
2. Откроется раздел настроек
3. **Проверьте что видите:**
   - Necessary - disabled, checked (всегда включен)
   - Analytics - **NEW** disabled, checked (всегда включен)
   - Marketing - обычный toggle (можно менять)
4. Измените Marketing (включите/отключите его)
5. Нажмите "Сохранить"
6. Проверьте: `getCookieData()`
7. **Ожидается:** `analytics: true`, `marketing: <ваш выбор>`

### Сценарий 5: Повторный визит (куки уже существуют)

1. Перезагрузите страницу (если куки установлены)
2. **Ожидается:** Окно согласия НЕ появляется
3. Проверьте: `getCookieData()`
4. **Ожидается:** Существующие куки загружены с `analytics: true`

### Сценарий 6: Открыть настройки извне

1. В консоли выполните: `window.openCookieSettings()`
2. **Ожидается:** Окно согласия открывается с включенной панелью настроек

## 🔍 Что проверить в продакшене

### 1. Яндекс.Метрика

- [ ] Откройте [Яндекс.Метрика](https://metrica.yandex.com/)
- [ ] Проверьте, что события отправляются (даже если пользователь отклонил маркетинг)
- [ ] События `reachGoal`: `cookie_modal_shown`, `cookie_accept_all`, `cookie_decline_all`, `cookie_save_settings`

**Код для проверки в консоли:**
```javascript
// Должна сработать (ym должна быть определена)
if (typeof ym === 'function') {
    ym(105770392, 'reachGoal', 'test_goal');
    console.log('✅ Яндекс.Метрика работает');
}
```

### 2. GA4

- [ ] Откройте [Google Analytics 4](https://analytics.google.com/)
- [ ] Проверьте Real-time события
- [ ] Должны быть события от пользователей

**Код для проверки в консоли:**
```javascript
if (typeof gtag === 'function') {
    gtag('event', 'test_event');
    console.log('✅ GA4 работает');
} else {
    console.log('⚠️ GA4 не загружена или не инициализирована');
}
```

- [ ] Проверьте что GTM контейнер загружается

**Код для проверки в консоли:**
```javascript
console.log('dataLayer:', window.dataLayer);
```

### 4. Marketing куки (если используете)

- [ ] Если у вас есть маркетинг платформы (Facebook Pixel, etc.)
- [ ] Проверьте что они загружаются ТОЛЬКО когда `marketing: true`

## ⚠️ Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| Окно согласия не появляется | Удалите куку: `deleteCookie()` и перезагрузитесь (F5) |
| analytics в куке = false | Это ошибка! Проверьте что используется новая версия js/cookie-consent.js |
| Яндекс.Метрика события не отправляются | Проверьте что ym функция загружена перед cookie-consent.js |
| window.__applyCookieConsent не вызывается | Убедитесь что функция определена ДО нажатия кнопок, см. COOKIE_CONSENT_HTML_EXAMPLE.html |
| Куки пустые после клика | Проверьте консоль на JavaScript ошибки (F12 → Console) |
