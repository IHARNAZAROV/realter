- `.agent-info-block` — блок информации об агенте

**Особенности:**
- Mobile-first дизайн
- Premium real estate стилистика
- Dark mode поддержка для footer
- Поддержка высокого контраста
- Smooth анимации (с учетом prefers-reduced-motion)

---

### 2. JavaScript: Consent Checkbox Manager (`/js/consent-checkbox.js`)

**Назначение:** Автоматическое добавление checkbox согласия на обработку ПД ко всем формам

**Функциональность:**
- Автоматически находит все формы на странице (booking, contact, inquiry)
- Добавляет обязательный checkbox согласия перед кнопкой отправки
- Валидация формы не пройдет без согласия
- Отслеживает динамически добавленные формы
- Ссылки на Политику конфиденциальности открываются в новых вкладках

**Использует:**
- Vanilla JS без зависимостей
- HTML5 атрибут `required` для валидации
- MutationObserver для отслеживания DOM изменений

**Селекторы форм:**
```
#viewing-booking-form
form[data-api-endpoint]
.contact-form
form[name="contact"]
form[name="booking"]
form[data-type="inquiry"]
```

## 🔍 Как проверить работу

### Проверка компонентов на странице объекта

1. Перейти на любую страницу объекта (`/objects/slug`)
2. В sidebar должна появиться новая информация об агенте
3. В конце основного контента (перед footer) должен быть блок с информацией об агентстве
4. В footer должна быть полная информация с УНП, лицензией, страховым полисом

### Проверка согласия на ПД

1. На любой странице со скролл вниз к footer
2. На странице объекта нажать "Записаться на просмотр"
3. В booking modal должен быть checkbox согласия
4. Попытаться отправить форму без согласия — должна показаться ошибка
5. После отметки checkbox форма должна отправляться

### Проверка на мобильных

1. Открыть на iPhone/Android
2. Все элементы должны быть видны и работать
3. Checkbox должен быть удобного размера для тача (20x20px)
4. Шрифты должны быть читаемыми

## ⚙️ Управление компонентами

### Изменение информации об агентстве

Отредактировать `/js/agency-legal-block.js`:
```javascript
const AGENCY_DATA = {
  name: 'Агентство недвижимости «ГермесГрупп»',
  unp: '193776843',
  license: '02240/487',
  licenseDate: '07.08.2024',
  insurance: { ... }
};
```

### Изменение текста согласия

Отредактировать `/js/consent-checkbox.js`:
```javascript
const CONSENT_TEMPLATE = `
  ... Нажимая кнопку, вы соглашаетесь с ...
`;
```

### Добавление новых форм

Добавить селектор в `/js/consent-checkbox.js`:
```javascript
const FORM_SELECTORS = [
  // ... существующие селекторы
  'form.my-new-form', // новая форма
];
```
