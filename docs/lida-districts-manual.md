# Как вручную задать корректные границы районов Лиды

Если текущие полигоны районов выглядят неточно, самый надёжный путь — нарисовать их вручную по реальным улицам/кварталам и сохранить в `data/lida-districts.geojson`.

## Что важно знать про формат

- Файл районов: `data/lida-districts.geojson`.
- В GeoJSON координаты должны быть в формате **[longitude, latitude]**.
- На карте Яндекс в коде они автоматически конвертируются в `[lat, lon]`, поэтому в файле нужно оставлять именно GeoJSON-формат.

## Рекомендуемый способ (самый удобный)

### 1) Откройте редактор полигонов

Подойдут:
- https://geojson.io
- https://yandex.ru/map-constructor (можно нарисовать там, затем экспортировать/перенести координаты)

### 2) Включите базовую карту и приблизьте Лиду

Ориентируйтесь по:
- крупным улицам,
- железной дороге,
- реке Лидейке,
- квартальной застройке.

### 3) Нарисуйте отдельный полигон для каждого района

Список районов и slug (должен совпадать):
- Центр → `centr`
- Старый Южный → `staryj-yuzhnyj`
- Южный Городок → `yuzhnyj-gorodok`
- Север → `sever`
- Рыбиновского → `rybinovskogo`
- Молодёжный → `molodezhnyj`
- Индустриальный → `industrialnyj`
- Пески → `peski`
- Слобода → `sloboda`
- Росляки → `roslyaki`
- Гастелло → `gastello`
- Заречье → `zareche`

### 4) Сохраните FeatureCollection в файл

Структура каждого района должна быть такой:

```json
{
  "type": "Feature",
  "properties": {
    "name": "Центр",
    "slug": "centr"
  },
  "geometry": {
    "type": "Polygon",
    "coordinates": [
      [
        [25.000000, 53.000000],
        [25.000000, 53.000000],
        [25.000000, 53.000000],
        [25.000000, 53.000000]
      ]
    ]
  }
}
```

## Проверка перед загрузкой

Выполните в проекте:

```bash
python -m json.tool data/lida-districts.geojson >/dev/null
```

Если команда без ошибок — JSON валиден.

## Частые ошибки

1. **Перепутан порядок координат** — нужно `[lon, lat]` в файле.
2. **Не закрыт полигон** — первая и последняя точка контура должны совпадать.
3. **Неверный slug** — из-за этого фильтр района не сработает.
4. **Самопересечение контура** — район может рисоваться некорректно.

## Быстрый рабочий процесс (рекомендую)

1. Отредактировали один район.
2. Обновили страницу.
3. Проверили hover/click и список объектов справа.
4. Перешли к следующему району.

Так проще поймать ошибки сразу, чем править все 12 районов за один раз.
